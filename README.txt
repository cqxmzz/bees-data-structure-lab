    控制台应用程序：bees_vs 项目概述

bees_vs.sln
    这是工程文件 需要打开

bees_vs.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 
    的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

bees_vs.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////

使用 VS2010
解决方案配置: Release        !!!(重要 IMPORTANT)!!!
解决方案平台: Win32

/////////////////////////////////////////////////////////////////////////////

测试函数已经在主应用程序源文件内集成

按F5运行
内容如下:

BEES TEST
BY CQM
1. functional test
2. performance test
3. compare to STL
4. QUIT


输入相应的数字, 并按回车以进入功能.
期间可能有等待, 直到需要按任意键继续(press any key to continue), 请耐心

其中对每个方法都进行了功能测试
对一部分进行了性能测试
还对一小部分性能和C++的STL进行了对比

[[OK]] 表示功能测试通过
[??]ms 表示运行时间为...毫秒
